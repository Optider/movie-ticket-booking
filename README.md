## Description

The APIs for a movie ticket booking portal.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## View API docs

Start the app and go to <http://localhost:3000/api>, to view swagger docs.
